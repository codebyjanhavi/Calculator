# 🧮 Calculator Web App

A sleek, responsive, and user-friendly calculator built with **HTML**, **CSS**, and **JavaScript**.

🔗 **Live Demo:** https://codebyjanhavi.github.io/Calculator/

<img src="images/calculator.png" alt="Calculator Screenshot" width="350"/>

</div>

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 📊 Percentage calculations
- 🔢 Decimal number support
- 🗑️ **AC** button to clear all input
- ⌫ **DE** button to delete the last character
- ⌨️ Full keyboard support
- 📱 Responsive design
- ⚠️ Error handling for invalid expressions

---

## ⌨️ Keyboard Shortcuts

| Key | Function |
|-----|----------|
| `0 - 9` | Enter numbers |
| `+` `-` `*` `/` `%` | Operators |
| `.` | Decimal point |
| `Enter` | Calculate result |
| `Backspace` | Delete last character |
| `Esc` | Clear all input |

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/codebyjanhavi/Calculator.git
```

### Navigate to the project folder

```bash
cd Calculator
```

### Run the project

Open `index.html` in your browser.

Or use the **Live Server** extension in VS Code.
